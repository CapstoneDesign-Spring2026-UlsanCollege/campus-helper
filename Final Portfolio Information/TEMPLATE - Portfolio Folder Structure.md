# Portfolio Folder Structure Template

Copy the starter folder into your team repository.

```txt
portfolio/
├── README.md
│
├── 01-project-overview/
│   ├── PROJECT_SUMMARY.md
│   ├── FINAL_MVP_SCOPE.md
│   └── SCOPE_DECISIONS.md

├── 02-semester-journey/
│   ├── SEMESTER_JOURNEY.md
│   ├── weekly-sprints/
│   │   ├── WEEK_01.md
│   │   ├── ...
│   │   └── WEEK_16.md
│   └── sprint-summaries/
│       ├── SPRINT_01.md
│       ├── SPRINT_02.md
│       ├── SPRINT_03.md
```

## Rules

- Rename files where easy.
- Link old files when moving would break evidence.
- Preserve all available versions.
- Do not invent missing historical files.
- Remove empty folders only when they are not useful.
- Make `portfolio/README.md` easy to navigate.
