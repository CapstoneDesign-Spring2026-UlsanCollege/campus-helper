# 3. Midterm Story Spine

**Problem**: Students need a centralized system for campus communication, academic tools, collaboration, and support services.  

**User**: Ulsan College students who need quick and organized access to campus resources and information.  

**Solution**: Campus Helper is a full-stack web platform that combines AI assistance, notes sharing, marketplace services, schedules, communication tools, authentication, and admin notices into one system.  

**Main demo flow**: Show a student logging in, accessing Marketplace and Notes, then demonstrate an admin publishing notices through the admin dashboard.  

**Proof**: Working React/Next.js frontend, MongoDB integration, JWT authentication, responsive UI improvements, backend API optimization, and updated project documentation.  

**Roadmap**: Improve performance, add more user interaction features, deploy the platform, strengthen testing, and prepare the final presentation.  

**Ask**: We want feedback on the clarity of the Midterm story, demo reliability, and overall usability of the platform.
