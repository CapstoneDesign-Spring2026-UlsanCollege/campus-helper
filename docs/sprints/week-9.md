# 3. Midterm Story Spine

**Problem**: Students need a centralized platform for communication, collaboration, academic support, and campus interaction.  

**User**: Ulsan College students who require quick and organized access to campus tools, resources, and student services.  

**Solution**: Campus Helper is a full-stack campus platform that combines AI assistance, notes sharing, marketplace services, authentication, schedules, communication tools, and admin notices into one integrated system.  

**Main demo flow**: Demonstrate a student logging in, interacting with Marketplace and Notes using the new Like feature, then show an admin publishing notices through the admin dashboard.  

**Proof**: Working frontend and backend integration, JWT authentication, database synchronization, Like feature implementation, responsive UI improvements, and updated documentation.  

**Roadmap**: Deploy the application, improve testing coverage, optimize performance further, and prepare final presentation materials and deployment proof.  

**Ask**: We want feedback on feature usability, demo clarity, and whether the project effectively demonstrates a real campus support workflow.
