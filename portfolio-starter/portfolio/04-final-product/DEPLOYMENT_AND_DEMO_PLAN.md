# Deployment and Demo Plan

## Primary Demo Plan

1. Open Campus Helper in a web browser.
2. Log in using a student account.
3. Demonstrate the Student Dashboard.
4. Open the Notes module and show note creation, viewing, and Like functionality.
5. Open the Marketplace module and show item listings and Like functionality.
6. Demonstrate the AI Assistant by asking a campus or study-related question.
7. Log out and log in using an admin account.
8. Open the Admin Dashboard.
9. Create and publish a campus notice.
10. Show that the notice appears on the platform.
11. Briefly explain the system architecture, database integration, and key project features.
